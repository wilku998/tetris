## Technologies
Project is created with:
* React: 16.8 (only function components and hooks)
* Redux: 4
* Styled-components: 4.2
* Typescript: 4.5.5
	
## Setup
To run this project, install it locally using npm:
* npm install
* npm run dev-server

To build:
* npm build

or
* npm build:prod
